Tutti

https://github.com/miscellus/Tutti.

Take a listen for yourself if you dare:

Tutti is a music generation library written in C99 that lets you have fun with sound synthesis and composition. The library generates PCM wave audio files. To test it out, I have created a crude rendition of Dr. Dre’s 2001 hit, Still D.R.E.

Oh, and it drives my wife nuts when I work on it without headphones.
